מדריך 1 בקיצור - הרעיון שמאחורי כל דבר

Full Stack

React מציג, Express מקבל בקשות, Mongo שומר, TypeScript מגדיר חוזה נתונים, Next.js מסדר אפליקציה מלאה.

JavaScript

const כברירת מחדל, objects הם reference, async רץ לפי sync -> microtasks -> macrotasks.

React

state מנהל זיכרון מקומי; hooks מחברים API/DOM/חישוב; forms ו-validation הם אזור ציון מרכזי.

מושגמשפט מבחן
useStateכל input וכל toggle צריכים state.
useEffectAPI/timer/listener, לא חישוב רגיל בתוך render.
Mongoose Modelדרך מסודרת לבצע CRUD מול Mongo.
Type Unionמאפשר narrowing לפי שדה מבדיל.

מדריך 2 בקיצור - פתרון שיעורי בית ומבחנים

  1. React Basics: components, props, map, form, state.
  2. Hooks/API: fetch, loading, error, filter, categories, ref focus.
  3. Budget TS: Transaction type, add/edit/delete, summaries.
  4. Express/Mongo: cors, express.json, schema/model, GET/POST/PUT/DELETE.
  5. Next.js: dashboard routes, layout, Tailwind, table/form/tabs.
  6. JS 20: input validation, clear Error, exact output object.
  7. TS 10: type/interface/enum/union/narrowing.

Stop Conditions

  • אם validation חסר - מתקנים לפני עיצוב.
  • אם נתקעת 10 דקות - מסמן TODO וממשיך.
  • אם API לא עובד - בודק URL, CORS, JSON, server running, status.
  • לא בונים פיצ׳ר שלא מופיע בדרישה.
  • לא משתמשים בנתונים מומצאים.
