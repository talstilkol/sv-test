# SVCollege Practical Guides

חבילת שלושת המדריכים המאוחדת למבחן Full Stack.

## איך לפתוח

פתח את הקובץ:

`index.html`

אם אתה לא יודע מאיפה להתחיל, פתח קודם:

`start_here.html`

ומשם עבור לאחד משלושת המדריכים:

1. `guide_01_ideas.html` - להבין את הרעיון שמאחורי כל דבר.
2. `guide_02_homework_and_exam_drills.html` - לפתור שיעורי בית ותרגילי מבחן.
3. `guide_03_exam_day_runbook.html` - לדעת מה לעשות בזמן המבחן.

לכרטיס חירום של עמוד אחד לפני המבחן:

`one_page_exam_cheatsheet.html`

לרשימת כל המשימות שנשארו קדימה עם טבלת זמנים:

`forward_tasks_plan.html`

להדפסה מרוכזת:

`print_pack.html`

לתוכנית יומית סגורה:

`study_path_37h45.html`

הערה: שם הקובץ נשאר לתאימות קישורים קיימים, אבל התוכן עודכן למסלול **100:20** אחרי הוספת עץ 73 סעיפי המבחן.

לסימון התקדמות:

`progress_tracker.html`

לבדיקת מצב החבילה:

`status.html`

## איך להשתמש

### אם אתה לא יודע מאיפה להתחיל

1. פתח את מדריך 1.
2. קרא את התמונה הגדולה ואת React/Backend/TypeScript.
3. עבור למדריך 2 ופתור תבנית אחת.
4. סיים עם מדריך 3 כדי לתרגל זמן מבחן.

### אם אתה כבר מבין בסיס אבל חלש בפרויקט

1. פתח את מדריך 2.
2. בצע לפי הסדר: React Basics, API Filter, Budget, Express/Mongo, Next.js.
3. בדוק כל פתרון עם checklist.
4. עבור למדריך 3 לסימולציה.

### אם המבחן קרוב מאוד

1. פתח את `one_page_exam_cheatsheet.html`.
2. עבור למדריך 3 רק אם צריך פירוט נוסף.
3. תרגל JS 20 ו-TS 10 מתוך מדריך 2.
4. חזור רק לנושאים שנכשלים ב-Gate.

### אם אתה צריך מסלול יומי

1. פתח את `study_path_37h45.html`.
2. בצע אבחון בסיס.
3. עבור יום-יום לפי ה-Gate.
4. ביום 7 סגור גם את עץ 73 סעיפי המבחן לפי Explain + Execute.
5. אל תעבור יום אם Gate היום לא ירוק.

### אם אתה רוצה לראות כמה נשאר

1. פתח את `progress_tracker.html`.
2. סמן רק משימות שבאמת בוצעו.
3. בדוק כמה שעות נשארו.
4. השתמש ב"ייצא התקדמות" כדי לשמור קובץ גיבוי.
5. השתמש ב"ייבא התקדמות" כדי לשחזר בדפדפן אחר.

## הדפסה

בכל מדריך יש כפתור הדפסה. מצב ההדפסה מנקה צבעים כהים ומסדר את העמוד לקריאה.

## מקורות

מפת המקורות נמצאת ב:

`source_map.md`

## כללי איכות

- אין שימוש באקראיות.
- אין fake data.
- צפייה ב-114 הסרטונים נמצאת בלוח המשימות לפי 30 דק׳ לסרטון; אין להסיק מהם חומר חדש בלי תמלול אמיתי.
- התוכן ממוקד במבחן Full Stack הקרוב: פרויקט 70, JavaScript 20, TypeScript 10.

## בדיקת תקינות

אפשר להריץ מתוך תיקיית החבילה:

```bash
node verify_guides.js
npm run guides:closed-route:strict
npm run guides:closed-route:browser-runtime:strict
npm run guides:release-gate:strict
npm run guides:strict:all
```

הבדיקה מוודאת:

- כל קובץ שמופיע ב-`manifest.json` קיים.
- כל קישור פנימי בין קבצי HTML תקין.
- לכל HTML יש כפתור הדפסה ו-CSS להדפסה.
- אין תבניות אסורות או placeholder.
- מסלול סגור עובד בכל עמוד (2 חצים, מפת שלבים, אחוז התקדמות, לוח יומי).
- בדיקת runtime בדפדפן מאשרת שהלוח היומי וההתראות נטענים בפועל.
- שער שחרור מאחד את כל דוחות המוכנות לדוח יחיד.

הפלט נשמר גם ב:

`verification_report.json`
`closed_route_smoke_report.json`
`closed_route_browser_runtime_report.json`
`guides_release_gate_report.json`
