7. שאלות JavaScript 20 נקודות

השאלות חוזרות סביב מערכים, אובייקטים, סכומים, ספירה, סדרות, תת-מערך, בדיקת קלט וזריקת שגיאה.

function sumByDigitLength(numbers) {
  if (!Array.isArray(numbers)) throw new Error("input must be an array");
  return numbers.reduce((result, number) => {
    if (typeof number !== "number" || Number.isNaN(number)) {
      throw new Error("all items must be numbers");
    }
    const key = String(Math.abs(number)).length;
    result[key] = (result[key] || 0) + number;
    return result;
  }, {});
}
  • בדוק שהקלט הוא מערך.
  • בדוק שכל איבר הוא מספר תקין.
  • החזר בדיוק את המבנה המבוקש.
  • אל תשנה את הקלט המקורי אם לא ביקשו.

User Union

type GuestUser = { kind: "guest"; visits: number };
type RegisteredUser = { kind: "registered"; email: string };
type User = GuestUser | RegisteredUser;

function labelUser(user: User) {
  if (user.kind === "registered") return user.email;
  return String(user.visits);
}
